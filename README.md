### **🤔 생각해볼 거리**
* bytes and KB
```
컴퓨터 과학에서는 이진 단위를 쓴다 -> 1000이 아닌 1024를 사용
그렇다면 bytes to KB는 1KB당 1024bytes로 계산함

하지만 하드웨어나 디스크의 크기를 나타낼 때는 -> 1000을 사용
이 경우에 bytes to KB는 1KB당 1000bytes로 계산함
```
